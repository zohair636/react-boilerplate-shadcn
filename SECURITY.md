# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability, please report it responsibly. Do not open public issues for security-related concerns.

Email: [hafiz.zohair2@gmail.com](mailto:hafiz.zohair2@gmail.com)

Please include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested fixes (if any)

I aim to acknowledge reports within 48 hours and follow up with next steps within 7 days.

## Supported Versions

Security updates are provided for the latest version of the boilerplate.

## Security Updates

Security fixes will be released as patch updates when necessary.

You are encouraged to:
- Keep dependencies up to date
- Follow best practices for securing React applications
